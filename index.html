What is RGB?

Every color can be represented using an RGB code, which stands for Red, Green, and Blue. Each number is between 0 and 255, indicating how much red, green, and blue are used to create the color. For example:

Red is (255, 0, 0)
Green is (0, 255, 0)
Blue is (0, 0, 255)
Yellow is (255, 255, 0)
Orange is (255, 165, 0)
